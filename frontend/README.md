# Frontend (React + Vite)

## Technology Stack

### Core Framework
- **React 18.3.1**: UI library with functional components and hooks
- **Vite 5.3.1**: Next-generation bundler with fast dev server
- **@vitejs/plugin-react 4.3.1**: React Fast Refresh plugin for Vite

### Routing & Navigation
- **React Router DOM 6.24.1**: Client-side routing with protected routes

### HTTP Communication
- **Axios 1.7.2**: Promise-based HTTP client for API requests

### Styling
- **CSS3**: Modern CSS with Grid, Flexbox, and CSS variables
- **Responsive Design**: Mobile-first approach with media queries

## Run locally

1. Install dependencies:
   - `npm install`
2. Copy `.env.example` to `.env` if needed.
3. Start dev server:
   - `npm run dev`

Default URL: `http://localhost:5173`
